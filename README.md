#  Monitor Estratégico Energético Latam
### Proyecto Integrador de Big Data & Visualización

Este Dashboard interactivo combina **Machine Learning** (Análisis de Sentimientos) con **Big Data** (Archivos Parquet) para analizar la infraestructura eléctrica en Brasil y Venezuela.

##  Funcionalidades
- **Extracción NoSQL:** Procesamiento de comentarios en formato JSON.
- **NLP (Sentiment Analysis):** Clasificación automática de opiniones en español.
- **Integración de Datos:** Cruce de información en tiempo real con datasets masivos en Parquet.
- **KPIs Ejecutivos:** Semáforos dinámicos para toma de decisiones.

##  Tecnologías
- Python 3.x
- Streamlit (Dashboarding)
- Pandas & PyArrow (Data Engineering)
- TextBlob (ML/NLP)
